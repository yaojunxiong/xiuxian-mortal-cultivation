# xiuxian-mortal-cultivation

静态网页版「凡人入门」任务体验，支持本地存档与台词导出。

## 怎么玩
1. 打开 `index.html`（GitHub Pages 或本地双击皆可）。
2. 主界面左侧显示当前章节、任务目标与 NPC 对话框，右侧提供区域按钮：
   - **进入山门**：触发 Q001「宗门登记」，点击「对话下一句」逐句阅读执事对白。
   - **进入后山采药区**：完成 Q001 后可用，采集灵草并用「返回外门区」交付完成 Q002。
   - **进入考核石台**：完成 Q002 后可用，在此选择「退让」或「硬拼」完成 Q003，查看章节结算与奖励。
   - **采集灵草 / 返回外门区**：位于采药区时可用，满足灵草数量后返回外门交付。
3. 对话与分支：
   - 使用「对话下一句」按顺序播放 NPC 台词；Q003 对话结束后解锁分支按钮。
   - 选择「退让」或「硬拼」都能结束考验，奖励功法《清风吐纳》，分支结果会在「章节结算」区显示。
4. 进度保存：页面自动将位置、任务、灵草计数等写入 `localStorage`，刷新不会丢失；点击「重置进度」可清空。
5. 导出剧情/台词：
   - 点击「导出剧情/台词」弹出文本框，直接复制即可；不会触发下载。
   - 同步文本保存在 `export/minecraft_npc.txt`，可复制到 MC Education 的 NPC 对话配置中。

## 数据文件
- `data/quests.json`：三段任务（宗门登记、采药试炼、初战考验）的目标、触发与完成条件。
- `data/dialogues.json`：按 NPC 整理的台词。
- `data/flags.json`：默认境界与提示，可自行扩展。

## 开发提示
纯静态网页，无需安装依赖，修改文件后直接刷新浏览器即可看到效果。
