# 离线修仙小筑

一个纯前端、离线友好的修仙主题笔记站点，所有数据都存放在仓库中的 JSON 文件，便于自托管与导出。

## 使用说明
- 打开仓库根目录下的 `index.html` 即可体验，无需后端。
- 数据位于 `data/` 文件夹，可直接编辑 JSON 文件更新页面内容。
- 点击“导出当前记事”按钮会生成简要的文本备份，方便保存。

## GitHub Pages 发布

## 目录结构
- `index.html`：主页入口。
- `style.css`：页面样式。
- `main.js`：数据读取与渲染逻辑。
- `data/`：默认演示数据。
- `export/`：可放置导出的文本或自定义备份。
